# Image Converters

Статический набор браузерных инструментов для работы с изображениями. Приложение открывается одной оболочкой с вкладками и не требует backend.

## Инструменты

- **ASCII** - конвертирует изображения в ASCII-графику, поддерживает очереди, пользовательские наборы символов, авто-контраст, дизеринг, TXT/PNG/ZIP экспорт.
- **PIXEL** - превращает изображения в ретро-пиксель-арт с палитрами, дизерингом, шумом, сканлайнами и PNG/ZIP экспортом.
- **RESIZE** - меняет размер и формат изображений, поддерживает качество сжатия, ресемплинг, сохранение пропорций и пакетную обработку.

## Структура

```text
index.html                 # Оболочка с вкладками
style.css                  # Тема и layout оболочки
projects/
  ascii-convert/           # ASCII-конвертер
  retro-pixel/             # Pixel-art конвертер
  image-resize/            # Resize & Compress
shared/
  css/                     # Общие панели, drag-drop, output, modal, scrollbars
  js/                      # Общая логика файлов, очередей, модалок и вкладок
mobile/                    # Адаптивные стили
```

## Тема

Интерфейс использует OLED/Archiver theme: черный фон, серые панели, оранжевый акцент `#F97316`, display-шрифт `Syne` и mono UI-шрифт `IBM Plex Mono` с fallback на системные monospace-шрифты.

Основные визуальные токены находятся в:

- `style.css`
- `shared/css/tool-core.css`

## Разработка

- Логика конвертации живет в `projects/*/app.js`.
- Общие UI-поведения и помощники живут в `shared/js/app-core.js`.
- Визуальные изменения лучше держать в CSS, не меняя DOM ID и классы, на которые подписан JavaScript.
- Перед завершением правок стоит запускать:

```bash
git diff --check
```
